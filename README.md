# SBI-Portfolio-Analyzer

SBIのポートフォリオページから保有銘柄の各種情報を取得して書き出します。


## 動作環境

- Windows
- Mac OS

## はじめに

ポートフォリオページの保有銘柄テーブルに下記データが表示されている必要があります。  
表示されていない場合は「表示形式」を変更してください。

- 銘柄（コード）
- 現在値
- 取得単価
- 数量
